defineErrorClass(arg0)

Defines a new error class.

Parameters:
Name Type Description
arg0 Object

The argument to be deconstructed.

Properties
Name Type Attributes Description
name string <optional>

A name for instances of this class. This argument must be present if arg0.code is missing, and the code becomes the upper-cased snake format of the name. For example, passing the name SomethingWickedError causes the code to be E_SOMETHING_WICKED.
code string <optional>

A code for instances of this class This argument must be present if arg0.name is missing. For example, passing the code E_SOMETHING_WICKED causes the name to be SomethingWickedError.
supererror * <optional>

An optional superclass previously returned by this function.
